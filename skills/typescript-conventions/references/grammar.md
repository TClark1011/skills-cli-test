# Grammar & Spelling

- All code (identifiers, comments, docs) must be written in English.
- Watch for misspellings, especially in function and variable names.
- If a word has US/British variants, match the variant already used in the codebase.

```typescript
// Bad — misspelled or non-English
const submited = true;
envoyerCode() {}
updateComapnyList() {}
setAcces() {}

// Good
const submitted = true;
submitCode() {}
updateCompanyList() {}
setAccess() {}
```
